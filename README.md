Simple ethereum pirvate network
-

Bootstrap ethereum network for development

Usage:

1. Initialize blockchain for node 01:

```bash
./eth-spn.sh initgeth 01
```

Or

```bash
./eth-spn.sh initparity 01
```

2. Bootup node 01: 

```bash
./eth-spn.sh upgeth 01
```

Or

```bash
./eth-spn.sh upparity 01
```
